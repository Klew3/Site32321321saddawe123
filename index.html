About Cartridge the Tiger

Features 🐯 Run, jump, swim, fly, carry, throw, and roll around with a custom retro physics engine. Designed for a controller. Includes a story mode. Built-in level editor. Create your own levels. Create your own stories. Built on a custom game engine designed from the ground up for modding. Includes an easy to use C# SDK, allowing anyone to create new objects for the level editor. Each month I'll post new content made using this SDK, including the source code. High framerate support. Resizable window. 🐯 Editor Features 🐯 Build on a map 20x bigger than Super Mario Maker. Build with no arbitrary limits on the number of objects. Select from over 200 objects, with over 25 enemies, or make your own. Place slopes, water, and decoration anywhere you want. Place signs with text to guide the player or tell a story. Place objects on different Z layers. Design complex puzzles using switches, wires, and logic blocks. Create atmosphere with interactive lighting. Control the camera with rooms. Mix and match different background layers. Add weather effects and post-processing. 🐯 Steam Workshop Features 🐯 Subscribe to new objects, background, foregrounds, stories, and other mods. Easily publish your own creations to the Steam Workshop. Package mods with stories to provide players with an entirely new experience. 🐯 Online Features 🐯 Upload levels and share them using a code. Browsing uploaded levels will be added post-release if there's enough demand. You do not need to create an account and the game does not require an internet connection.

Gameplay

Cartridge the Tiger is a pixel art platformer with a built-in level editor and C# SDK. I wanted to create something that exists somewhere between a game and a game engine, a toy box that can be enjoyed by casual players, kaizo players, level designers, people learning how to code, developers, and everyone in-between.
